# To Do list App
Una semplice ma funzionale applicazioneweb per gestire le attività quotidiane, sviluppata per esercitare la manipolazione del Dom e la gestione dello stato in JavaScript.

## Tecnologie utilizzate:
**HTML5**: struttura della pagina:
**CSS3**: layout e stile responsive.
**JavaScript (ES6)**: logica applicativa e gestioneeventi.

**Funzionalità**
**Aggiunta task**: inserimento dinamico di nuove attività.
**Rimoizone**: possibilità di eliminare i task completati.
**Design pulito**: interfaccia user-friendly e moderna.
